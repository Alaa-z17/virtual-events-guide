# دليل فعاليات الجامعة الافتراضية السورية (SVU Events Guide) 🎓

موقع إلكتروني متكامل مخصص لإدارة وعرض الفعاليات والأنشطة الجامعية، تم تطويره كجزء من المتطلبات العملية لمادة **تطوير الويب (BWP401)**.

## 📝 وصف المشروع (Project Description)
يهدف هذا المشروع إلى توفير منصة مركزية لطلاب الجامعة الافتراضية السورية للاطلاع على كافة الورشات التدريبية، الندوات الأكاديمية، والفعاليات الاجتماعية. يتميز الموقع بتصميم عصري ومتجاوب يضمن تجربة مستخدم سلسة على مختلف الأجهزة.

## ✨ المميزات الرئيسية (Key Features)
- **واجهة مستخدم متجاوبة:** باستخدام Bootstrap 5 لضمان التوافق مع الهواتف والأجهزة اللوحية.
- **عرض ديناميكي للفعاليات:** صفحة تفاصيل موحدة تتغير محتوياتها برمجياً بناءً على اختيار المستخدم.
- **نظام تحقق (Validation):** نموذج "اتصل بنا" مزود ببرمجة JavaScript للتحقق من البيانات قبل الإرسال.
- **سلايدر تفاعلي (Carousel):** لعرض أبرز الفعاليات الأسبوعية بشكل جذاب.
- **قوائم مخصصة:** أهداف ورؤية المشروع معروضة بتنسيق بصري فريد.

## 🛠 التقنيات المستخدمة (Technologies Used)
- **HTML5:** لبناء هيكلية الصفحات.
- **CSS3:** للتنسيقات المخصصة (Custom Styles).
- **Bootstrap 5 (RTL):** للإطار البرمجي المتجاوب ودعم اللغة العربية.
- **JavaScript (Vanilla JS):** للتحقق من المدخلات والتحكم بديناميكية المحتوى.
